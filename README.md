**ClassPulse – Real-Time Classroom Feedback and AI Quiz Platform**

Developed a cross-platform classroom engagement system consisting of a React Native mobile application for teachers and a web interface for students. The platform enables teachers to create live classroom sessions while allowing students to anonymously provide real-time feedback on their understanding of concepts through dynamic voting mechanisms.

Implemented real-time communication to instantly aggregate and visualize student responses, helping educators identify knowledge gaps and adjust their teaching approach during lectures. Integrated AI-powered quiz generation using OpenRouter APIs to automatically create assessment questions based on classroom topics, reducing manual effort and enhancing student engagement.

Designed the system to promote anonymous participation, increase classroom interaction, and provide data-driven insights into student comprehension.

**Tech Stack:** React Native, FireBase, WebSockets/Socket.IO, OpenRouter API
